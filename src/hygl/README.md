# 介绍
HoYoGameLauncher提供了一个米哈游聚合启动器，可以快捷的启动：
- 崩坏：星穹铁道
- 原神
- 原神国际服（暂未实现）
- 崩坏：星穹铁道国际服（暂未实现）
- 崩坏3（暂未实现）
